#  Movie API Web App

A web application that allows users to search for movies, view details, and manage a list of favorite films using the **OMDb API**.

This project was built to practice **frontend JavaScript**, **API integration**, and **UI state management**.

---

##  Features

-  Search for movies by title
-  View movie details (poster, plot, ratings, year, etc.)
-  Add and remove movies from a favorites list
-  Client-side state management (no backend)
-  Responsive design (desktop & mobile friendly)
-  Error handling for invalid or empty searches

---

##  Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **OMDb REST API**
- **Fetch API**
- **GitHub Pages** (for deployment)

---

##  How It Works

1. User enters a movie title
2. The app sends a request to the OMDb API
3. Movie data is returned as JSON
4. The UI dynamically updates using JavaScript
5. Users can save movies to a favorites section

---

##  API Used

- **OMDb API**  
  https://www.omdbapi.com/

> Note: An API key is required to make requests.
